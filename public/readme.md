## About Project
This is a Node/Express app for jotting down ideas for future Youtube videos

### It uses :=

- Nodejs
- Handlebars
- Bootstrap
- MongoDb

## Set up 

### clone the repository
```
git clone https://github.com/Kajal-Negi/vidjot.git
```
### change the directory
```
cd vidjot
```
### install the dependencies


```
npm install
```
